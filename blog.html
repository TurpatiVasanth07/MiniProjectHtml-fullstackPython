1. Variables and Data Types

Python supports multiple data types such as integers, floats, strings, and booleans:

x = 10
name = "Alice"
pi = 3.14
is_valid = True

2. Loops in Python

Python provides both for and while loops:

for i in range(5):
    print(i)

while True:
    break

3. OOP: Classes and Objects

Object-Oriented Programming helps structure your Python code:

class Person:
    def __init__(self, name):
        self.name = name

    def greet(self):
        print("Hello", self.name)

p = Person("Alice")
p.greet()

4. Comparison of Python Data Types

This table summarizes the common data types:

Type Example Description
int 10 Whole number
float 3.14 Decimal number
str "Hello" Text string
bool True / False Boolean value

5. Functions in Python

Functions allow you to reuse code:

def add(a, b):
    return a + b

print(add(3, 5))

6. File Handling Basics

Open, read, and write to files using open():

with open("example.txt", "w") as file:
    file.write("Hello file!")
